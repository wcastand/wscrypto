# Orderbook

Orderbook for XBT/USD Futures
production hosted on vercel (production)[https://wscrypto.vercel.app/]

## Install

```bash
yarn
```

## Usage (local dev)

```bash
yarn dev
```

## Build and serve locally the production

```bash
yarn build
yarn serve
```

## Run tests

```bash
yarn test
```

in watch mode

```bash
yarn test --watch
```
